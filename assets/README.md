# Assets

该目录用于存放星语音乐盒客户端相关素材：

- App 图标
- 截图
- 品牌色
- 设计草图
- 发布素材

视觉方向参考：

- 主 App 图标优先采用“星星音乐盒 / 水晶罩 / 月光湖面”意象。
- 当前 App 图标原始图为“仲夏星光”版本，Xcode 工程内 AppIcon 由 1024px 原图缩放生成，不额外添加圆角。
- 默认皮肤方向可围绕“仲夏星河”或“月泊银声”展开。
- 音符、声波、星星、麦克风、文件夹等元素可发展为小图标体系。

建议后续按用途拆分子目录：

- `app-icon/`
- `theme-midsummer-starry-river/`
- `theme-moonlit-silver-sound/`
- `icons/`
- `screenshots/`

素材如果来自 AI 生成、商业插画或第三方授权，应在对应目录记录来源、授权状态和可使用范围。
