# 星语音乐盒视觉方向

本文档记录星语音乐盒客户端的视觉方向，用于后续 App 图标、默认皮肤、设置页入口图标、空状态插画与发布素材设计。

当前阶段只沉淀设计方向与素材使用原则，不直接替换 Xcode 工程内的 AppIcon，也不承诺发布素材已经完成。

## 核心意象

星语音乐盒的主视觉优先围绕以下意象展开：

- 星星音乐盒
- 透明水晶罩
- 月光湖面
- 星光与流星
- 淡蓝、淡青、银白的夜色
- 安静、清澈、带一点童话感的音乐陪伴

其中“星星音乐盒 / 水晶罩 / 月光湖面”适合作为主 App 图标方向。它和“Music Box”的产品名贴合，识别度高，也能和后续多端客户端保持一致。

## App 图标方向

主 App 图标建议采用“水晶罩中的星星音乐盒”作为核心构图：

- 图标主体：水晶罩、星星、音乐盒底座。
- 背景：月光湖面、远山、星空。
- 色彩：淡蓝、淡青、银白为主，少量星光高亮。
- 观感：清澈、柔和、安静，但需要保证小尺寸下仍能识别出星星和音乐盒。

注意事项：

- 不直接使用过于复杂的大场景作为小尺寸 App 图标，避免缩小后糊成一团。
- 人物插画可以作为欢迎页、空状态或宣传素材方向，但主 App 图标优先使用“星星音乐盒”。
- 正式落入 Xcode 前，需要提供 1024x1024 原图，并按 iOS / iPadOS AppIcon 规格生成完整切图。

## 默认皮肤方向

默认皮肤可暂定为以下两个候选名称之一：

- 仲夏星河
- 月泊银声

主题元素建议：

- 主色：月光蓝、湖面蓝、淡青。
- 辅色：银白、星光白、浅花蓝。
- 强调色：柔和青蓝或偏银的高亮色，用于播放进度、当前歌词、选中状态。
- 背景：星空、湖面、月光、轻雾、微弱流星。

该主题应服务于播放器体验：安静、沉浸、不过度抢封面和歌词的注意力。

## 小图标体系

底部那组音符、声波、星星、麦克风、文件夹等元素，可以发展为一套轻量图标体系：

- README section 图标
- 设置页入口图标
- 空状态插画
- 播放列表、歌词、收藏、主题皮肤等功能入口图标
- 后续发布素材中的功能说明插图

小图标建议保持线性、圆润、低对比度，和默认皮肤的柔和夜色统一。

## 素材归档建议

后续素材建议放入 `assets/` 下，并按用途分组：

- `assets/app-icon/`: App 图标原图、切图、导出说明。
- `assets/theme-midsummer-starry-river/`: “仲夏星河”主题素材。
- `assets/theme-moonlit-silver-sound/`: “月泊银声”主题素材。
- `assets/icons/`: 小图标体系。
- `assets/screenshots/`: README、PR、发布说明使用的截图。

如果素材来自 AI 生成、商业插画或第三方授权，需要在素材目录内记录来源、授权状态和可使用范围。
