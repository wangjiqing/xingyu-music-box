# 在线音频播放预研

v0.4.2 不实现在线读取星语音库音频。后续版本开始前需确认：

- 星语音库提供受 HMAC 保护的音频流接口。
- 接口支持 HTTP Range，便于拖动、断点缓冲和弱网恢复。
- 服务端返回正确的 `Content-Type`、`Content-Length`、`Accept-Ranges`。
- 移动端 AVPlayer 对认证头、重试、重定向和 Range 请求的适配策略。
- 在线音库播放源与本地媒体库播放源的统一抽象。
- 网络异常、弱网、缓存、离线下载、后台播放和锁屏控制策略。
- 音频流接口的权限边界和日志脱敏。
- 服务器侧带宽、转码和 NAS 文件读取压力。
